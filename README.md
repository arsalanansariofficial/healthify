# Healthify

Healthify is a modern web-based platform that allows users to register, search for doctors, and book appointments based on their needs.

It includes secure authentication, role-based access control, and an admin-friendly dashboardâ€”built with **Next.js**, **NextAuth**, **Prisma**, **TailwindCSS** and **ShadCN**.

### Features

- Dashboards for Admin
- Appointment Scheduling
- Doctor Search & Filtering
- Fully typed with TypeScript
- Authentication & Authorization (NextAuth v5)
- Role-Based Access Control (Admin / Doctor / User)
- Modern UI built with **TailwindCSS** and **ShadCN**

### Requirements

- Node.js 20+
- MongoDB (depending on your Prisma setup)
- A configured .env file (see .env.example)

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Production Build

```bash
npm run build
npm start
```

### Environment Variables

Use the .env.example file to configure:

- NextAuth configuration
- Database connection URL
- Email provider credentials

### Database Import/Export

#### Export Database

```bash
npm run mongo:export
```

#### Import Database

```bash
npm run mongo:import
```
